### Features
* **Interactieve Metrics:** Cirkeldiagrammen (Gauges) die via JavaScript animeren van 0% naar het actuele niveau.
* **Tooltips:** Zwevende details bij mouse-over op de skill-kaarten voor extra context.
* **Responsive Design:** Volledig schaalbaar via Flexbox en CSS Grid (werkt op desktop en mobiel).
* **Dark Mode UI:** Moderne uitstraling met specifieke kleurcodering per domein (Groen voor ERP, Paars voor AI/Labs, etc.).
* **Privacy & Security:** Voorzien van `noai` meta-tags en `robots.txt` instructies om AI-scraping te beperken.
